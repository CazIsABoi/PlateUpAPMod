How to install
### How to install
1. Download the mod on the workshop
2. Launch the game and go into HQ (Lobby)
3. In Options > PreferenceSystem > PlateupAP, there you can make the config file and connect

### Known Issues
- No connection in main menu
- Sometimes day leases break
- Incorrect day counting for day goal at times
