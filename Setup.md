## How to install
1. Subscribe to the mod on the Steam Workshop \[RECOMMENDED] or if downloaded on GitHub, put the unzipped mod folder in the Mods folder (C:\\Program Files (x86)\\Steam\\steamapps\\common\\PlateUp\\PlateUp\\Mods) 
2. Launch the game and create the config in Options > PreferenceSystem > PlateupAP, there you can make the config file
3. Open the Config, Do windows + r and enter in "%appdata%\\..\\LocalLow\\It's Happening\\PlateUp"

4. If not done already create a yaml and create the multiworld, to add a speed upgrade to the start inventory add it with "Speed Upgrade Player: 1" and change the 1 to anything from 1-5

5. Connect to the multiworld by changing the values in the config and press "Connect"

Have fun! :D
